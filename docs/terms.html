2. Description of Service

NetFlow is a free, open-source Android application that provides the following functionality, all performed exclusively on-device:

  • Local VPN Proxy: Uses the Android VpnService API to act as a local transparent proxy, capturing and analyzing network traffic metadata (connection headers, IP addresses, ports, byte counts) without decrypting TLS payloads or routing traffic to external servers.
  • Per-App Traffic Analytics: Maps network connections to specific installed applications using Linux network socket UID mapping.
  • Risk Prediction: Scores connections against a locally stored list of known malicious hosts, suspicious port patterns, and tracker endpoints.
  • Ad & Tracker Blocking: Intercepts and blocks DNS requests and TCP connections to endpoints on a locally maintained blocklist.
  • Bandwidth Monitoring: Displays real-time and historical data usage statistics per application.

NetFlow does not operate any backend servers, collect user data, or route user traffic through remote infrastructure. All processing is performed in memory on the local device and discarded after use.
